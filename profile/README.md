# Diakonie RKN

Willkommen im GitHub-Organisationsprofil der **Diakonie Rhein-Kreis Neuss (Diakonie RKN)**.

Die Diakonie RKN engagiert sich für Menschen in unterschiedlichen Lebenslagen und bietet soziale Unterstützung, Beratung und Begleitung an. Unser Ziel ist es, Teilhabe zu stärken, Hilfe zugänglich zu machen und Menschen mit Respekt, Würde und Verlässlichkeit zu begegnen.

Weitere Informationen über uns finden Sie hier:
- https://www.diakonie-rkn.de/wir/
